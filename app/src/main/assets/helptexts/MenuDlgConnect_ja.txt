** 接続方法 **
"対面" : 二人がこの一つの端末上で対戦する
"Bluetooth" : ２つの端末を Bluetooth で接続して対戦
"IP" : インターネットプロトコルの略で、無線LANルーターなど、
WiFiアクセスポイントに接続して対戦。IPアドレスを指定して接続します
"WiFiダイレクト" : IP接続の一種ですが、アクセスポイントなしで端末同士を直接に接続
"NFC-Blutooth" : NFC をサポートする端末では NFCタグ部を
  10cm 以内に近づけて Bluetooth接続し対戦します
"NFC-WiFiダイレクト" : NFC をサポートする端末では NFCタグ部を
  10cm 以内に近づけてWi-Fiダイレクト接続し対戦します
----------------
Bluetoothの方が使い易い様ですが、問題があるようならWifiダイレクトを選んでください。
(NFC)Bluetooth と IP と (NFC)WiFiダイレクト を切り替えるとき、接続先の端末を変更するときは、
一旦現在の接続を切断してください。
いずれの場合も最初に行われる、近くの端末の"探索"には多少時間がかかります。

BluetoothにはレガシーモードとBLE(省電力モード)がありますが、
このアプリはレガシーモードです。BLEのみをサポートする端末では使用できません。
Wifiダイレクト を試してください。
また、電話機能がある場合は干渉が発生する可能性があります。
デュアルバンド(帯域が2.4GHz+5GHz)の端末の場合は電話の方は5GHzにしてください。
Bluetoothは2.4GHzを使用しています。
IP接続はベースの通信方法に起因する送信遅延が出る可能性があります。
（通信障害の検知は一定時間の遅れがでることがあります。）

Android6.0(MarchMallow)以降では最初にWifiダイレクトの使用の許可を求めるダイアログがでます。許可を与えてください。
